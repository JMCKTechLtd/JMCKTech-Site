Lab 1 – Getting Started with the Windows 11 Desktop

Welcome to your first Windows 11 lab.
In this lab, we’re just getting comfortable with the desktop.

You’ll explore the Start Menu,
the taskbar,
and the system tray.

Nothing complicated here.
No pressure.

The goal is simple:
recognise where things live
and feel at home on the screen.

If you’ve never used Windows before, that’s fine.
And if you have — this is still worth doing.

Let’s get started.
