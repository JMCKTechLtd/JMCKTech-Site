# BlockShell Lite Assets

This directory contains all visual assets for BlockShell Lite.

## Required Files:

### Root directory:
- blockshell-lite-logo.svg      # Main application logo
- icon-powershell.svg           # PowerShell technology icon

### module-icons/ directory:
- icon-core.svg                 # Core PowerShell module icon
- icon-ad.svg                   # Active Directory module icon  
- icon-exchange.svg            # Exchange & M365 module icon
- icon-server.svg              # Windows Server module icon
- icon-networking.svg          # Networking module icon
- icon-sql.svg                 # SQL Server module icon

## Icon Specifications:
- Size: 20x20 pixels minimum, 32x32 recommended
- Format: SVG (scalable vector graphics)
- Style: Flat, monochromatic or duotone
- Colors should match module color scheme
