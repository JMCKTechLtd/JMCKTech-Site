Lab Guide

Learning Objectives

In this lab, you will learn to:

  • Select multiple tickets using checkboxes
  • Assign tickets to team members
  • Reassign tickets to different groups
  • Escalate high-priority tickets
  • Track escalated ticket status

Lab Requirements

○ Select tickets for assignment
○ Open assignment modal
○ Complete ticket assignment
○ Open escalation modal
○ Complete ticket escalation

1 Select Tickets Current Step

Select one or more tickets using the checkboxes in the list.

2 Open Assignment Pending

Click the "Assign" button to open the assignment modal.

3 Assign to Team Member Pending

Select a team member from the dropdown and complete the assignment.

4 Select for Escalation Pending

Select a high-priority ticket that needs escalation.

5 Escalate Ticket Pending

Use the escalate button to escalate the ticket to senior staff.
