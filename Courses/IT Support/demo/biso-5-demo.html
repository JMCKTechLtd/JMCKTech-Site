Lab Guide

Learning Objectives

In this lab, you will learn to:

  • Navigate BIOS Advanced menu
  • Access CPU Configuration settings
  • Locate Virtualization Technology option
  • Enable Intel VT-x / AMD-V
  • Save BIOS configuration

Why this matters: Virtualization Technology is required to run virtual machines (VMs) with VMware, VirtualBox, Hyper-V, and KVM.

Lab Requirements

○ Navigate to Advanced menu
○ Enter CPU Configuration
○ Navigate to Virtualization setting
○ Enable Virtualization Technology
○ Save and exit BIOS

1 Navigate to Advanced Current Step

Use arrow keys to navigate to the "Advanced" menu in the left panel.

2 Enter CPU Configuration Pending

Press Enter to access the CPU Configuration submenu.

3 Enable Virtualization Pending

Navigate to "Intel Virtualization Technology" and press Enter to enable it.

4 Save Changes Pending

Press F10 to save changes and exit BIOS setup.
