Lab Guide

Learning Objectives

In this lab, you will learn to:

  • Navigate BIOS Setup Utility interface
  • Access Boot Settings menu
  • Change boot device priority order
  • Save BIOS configuration changes
  • Exit BIOS setup properly

Lab Requirements

○ Navigate to Boot Settings
○ Select Hard Drive device
○ Promote Hard Drive to first
○ Verify boot order is correct
○ Save with F10 and exit

1 Access Boot Settings Current Step

Navigate to the Boot Settings menu in the BIOS. This is already selected in the left menu panel.

2 Navigate to Hard Drive Pending

Use the Down Arrow (↓) key to navigate through the boot devices until Hard Drive is selected.

3 Promote Hard Drive Pending

Press Enter to promote the Hard Drive device up in the boot priority order. Repeat until it's first.

4 Verify Boot Order Pending

Confirm that Hard Drive is now listed as the first boot device (position 1).

5 Save and Exit Pending

Press F10 to save the changes and exit the BIOS Setup Utility.
