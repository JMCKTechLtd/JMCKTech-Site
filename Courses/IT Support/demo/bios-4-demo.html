Lab Guide

Learning Objectives

In this lab, you will learn to:

  • Navigate BIOS Boot settings
  • Switch from Legacy to UEFI boot mode
  • Enable Secure Boot (UEFI only)
  • Configure boot device priority
  • Understand UEFI advantages

Key Difference: UEFI supports Secure Boot, GPT partitions, and faster boot times compared to Legacy BIOS.

Lab Requirements

○ Navigate to Boot settings
○ Change to UEFI boot mode
○ Enable Secure Boot
○ Set Windows as 1st boot device
○ Save BIOS configuration

1 Change Boot Mode Current Step

Navigate to "Boot Mode" setting and press Enter to change from Legacy to UEFI.

2 Enable Secure Boot Pending

Navigate to "Secure Boot" and enable it (only available in UEFI mode).

3 Set Boot Priority Pending

Navigate to boot devices list and move Windows Boot Manager to position 1 using + key.

4 Save Changes Pending

Press F10 to save the configuration and exit BIOS.
