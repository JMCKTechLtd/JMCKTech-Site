Lab: Entra ID - Assign Microsoft 365 License
Goal: Assign a Microsoft 365 Business Premium license to Jane Smith.
  1. Navigate to Azure Active Directory admin center from Azure portal
  2. Go to Users > All users in the left navigation
  3. Search for and select Jane Smith
  4. Click Licenses in the user profile
  5. Select Microsoft 365 Business Premium license
  6. Click Assign license and confirm the success notification
