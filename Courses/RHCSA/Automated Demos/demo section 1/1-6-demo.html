1 Create a Tar Archive Current Step

First, create a tar archive containing two files.

Command to run in the real lab:

Understanding Tar Archives

What Combines multiple files into a single archive
Why Easier to move, copy, and back up groups of files
Flags -c create, -v verbose, -f file
Tip Order matters: tar -cvf archive.tar files‚Ä¶

How it works: tar walks over each file and writes it into a single archive file. No compression yet ‚Äì just packaging.

2 List Archive Contents Pending

Next, list the contents of the archive to see what it contains.

Command to run:

Tar Listing Mode

-t List archive contents instead of creating/extracting
Use Case Verify which files are in the archive before extracting

3 Extract Archive to a Directory Pending

Create a directory and extract the archive into it.

Commands to run:
mkdir extracted
tar -xvf archive.tar -C extracted

Extracting with tar

-x Extract files from an archive
-C dir Change to directory dir before extracting

Best practice: Extract into a dedicated directory so you don‚Äôt clutter your current working directory.

4 Compress and Decompress with gzip Pending

Now compress the tar archive with gzip and then decompress it again.

Commands to run:
gzip archive.tar
gunzip archive.tar.gz

gzip Basics

gzip Compresses a single file (replaces it with .gz version)
gunzip Reverses gzip compression
Why tar + gzip? First package files (tar), then compress the archive (gzip)

5 Create Compressed Tar Archives (gzip, bzip2, xz) Pending

Finally, create compressed tar archives using different algorithms in a single step.

Commands to run:
tar -czvf archive.tar.gz file1.txt file2.txt
tar -cjvf archive.tar.bz2 file1.txt file2.txt
tar -cJvf archive.tar.xz file1.txt file2.txt

Understanding Compression Options

-z (gzip) Fast, widely used, good default
-j (bzip2) Better compression than gzip, slower
-J (xz) Best compression, slowest
Choose Based on performance vs. size requirements

How it works: tar packages and compresses in one step, producing files like:

gzip: archive.tar.gz
bzip2: archive.tar.bz2
xz: archive.tar.xz
