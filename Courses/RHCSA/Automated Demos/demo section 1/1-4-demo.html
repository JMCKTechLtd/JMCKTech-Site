1 Listing Files with ls Current Step

First, let's explore the contents of your current directory using the ls command.

Type these commands exactly:
ls -l
ls -a

Understanding ls

What Lists directory contents
Why View files and directories
Common Options -l (long format), -a (show hidden)
Output Meaning Permissions, owner, size, date

How it works: ls reads the contents of a directory and prints them. ls -l shows detailed information, while ls -a includes hidden files (starting with .). Combine options like ls -la.

2 Creating and Copying Files Pending

Now let's create a new file and make copies using touch and cp.

Type these commands exactly:
touch newfile.txt
cp newfile.txt copied_file.txt
cp -r documents documents_backup

Understanding touch and cp

touch Creates empty files or updates timestamps
cp Copies files or directories
Options -r (recursive copy for directories)
Best Practice Use meaningful names; be careful with overwrites

How it works: touch creates a new empty file if it doesn't exist. cp source destination copies files, and cp -r is required to copy directories and their contents.

3 Moving and Renaming Files Pending

Next, let's move files into directories and rename them using mv.

Type these commands exactly:
mv copied_file.txt documents/
mv documents/copied_file.txt documents/renamed_file.txt

Understanding mv

What Moves or renames files
Why Organize files into directories and rename them
Examples mv file.txt dir/, mv old new
Tip Use full paths to avoid confusion

How it works: mv source destination moves a file or directory to a new location or renames it. When the destination is a directory, the source is moved inside. When the destination is a filename, the source is renamed.

4 Removing Files and Directories Pending

Now let's learn how to remove files and directories using the rm command.

Type these commands exactly:
rm file1.txt
rm -r documents_backup
Warning: rm permanently deletes files. Be careful, especially with rm -r and rm -rf.

Understanding rm

What Removes files or directories
Options -r (recursive), -f (force)
Best Practice Double-check paths before running
Tip Use ls to verify before deleting

How it works: rm filename removes a file. rm -r directory removes a directory and its contents recursively. Combine with -f to suppress prompts (dangerous if misused).

5 Finding Files Pending

Finally, let's learn how to find files using the find command.

Type these commands exactly:
find . -name "*.txt"
find /etc -type f -name "*.conf"

Understanding find Command

What Searches for files/directories
Power Can search by name, type, size, time, etc.
Examples find . -name "*.log", find /var -type d
Tip Combine with commands like -exec for powerful operations

How it works: find path options walks the directory tree starting at path and matches files based on the options you provide. You can search by name, type, permissions, size, and more.
