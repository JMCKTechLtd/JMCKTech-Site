5 Filtering by Service Current Step

Now filter the journal by the SSH service.

Type the command shown below:

journalctl -u sshd

Common journalctl filters:

  • -p: Priority (emerg, alert, crit, err, warning, notice, info, debug)
  • -u: Unit/service name
  • --since/--until: Time ranges
  • -k: Kernel messages only
