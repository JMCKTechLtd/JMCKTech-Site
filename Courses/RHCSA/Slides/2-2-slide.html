Common Targets
  • graphical.target - GUI with networking (runlevel 5)
  • multi-user.target - CLI with networking (runlevel 3)
  • rescue.target - Single-user mode (runlevel 1)
  • emergency.target - Emergency shell (most minimal)

Understanding Rescue Mode

Rescue mode provides a single-user root shell with minimal services running. In a real system, this would actually switch modes.

Warning: In a real system, rescue mode would require authentication and would actually change your system state.
