5 Understanding umask Current Step

Finally, let's understand and use the umask command.

umask
umask 0022
What Default permission mask
Why Sets default permissions for new files
Where Shell environment setting
When Affects all new file creations

How it works: umask subtracts permissions from the default (666 for files, 777 for directories). 0022 results in 644 (rw-r--r--) for files and 755 (rwxr-xr-x) for directories.

[user@lab ~]$ sudo chgrp wheel /var/tmp/permission_lab/file3.txt [user@lab ~]$ ls -l /var/tmp/permission_lab/file3.txt -rw------- 1 user wheel 1024 Jan 10 14:30 file3.txt Step 4 completed. Group changed to wheel. [user@lab ~]$ Next command: [user@lab ~]$ umask [user@lab ~]$ Then: umask 0022
[user@lab ~]$

6 Using umask Current Step

Notice how umask affects the default permissions of newly created files.

With umask 0022:

  • Files: 666 - 022 = 644 (rw-r--r--)
  • Directories: 777 - 022 = 755 (rwxr-xr-x)
[user@lab ~]$ umask 0022 [user@lab ~]$ umask 0022 Step 5 completed. Umask set to 0022. The umask value determines default permissions for new files and directories.
