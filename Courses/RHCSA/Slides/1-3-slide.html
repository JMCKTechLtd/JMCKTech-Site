5 Regular Expression Reference Current Step

Review the key regular expression concepts you've learned.

Pattern Meaning Example
^ Start of line ^2023
$ End of line failed$
[0-9] Any digit [0-9]%
[AEIOU] Any uppercase vowel [AEIOU]
| Alternation (OR) error|warning
{n} Exactly n occurrences [0-9]{3}

6 grep Options Reference Current Step

Review the key grep options you've learned.

Option Description Example
-i Case-insensitive search grep -i "hello"
-v Invert match (non-matching lines) grep -v "success"
-n Show line numbers grep -n "error"
-E Extended regular expressions grep -E "error|warning"

Understanding grep Basics

Why To find specific information in files
Where Works on text files or command output
