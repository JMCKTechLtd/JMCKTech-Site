Welcome to the RHCSA File Search and Copy Lab. In this lab, you will search for files owned by a specific user and copy them to a target directory.

Follow the steps in order. You must complete each step before moving to the next one.

1 Create Target Directory Current Step

First, let's create the target directory where we'll copy the files.

Type this command exactly:
mkdir -p /root/findresults

The -p option ensures parent directories are created if needed.

Understanding Directory Creation

The mkdir command creates directories. The -p flag creates parent directories as needed and doesn't throw an error if the directory already exists.

Creating the directory first ensures we have a place to copy the files we find.

2 Find Files Owned by User 'ira' Pending

Now let's search for all files owned by user 'ira' on the system.

Type this command exactly:
find / -user ira -type f 2>/dev/null

This command will search the entire filesystem for files (-type f) owned by user 'ira'.

The 2>/dev/null part suppresses permission denied errors.

Understanding the find Command

The find command is powerful for searching files based on various criteria:

  • / - Start search from root directory
  • -user ira - Find files owned by user 'ira'
  • -type f - Only search for regular files (not directories)
  • 2>/dev/null - Redirect error messages to null device

3 Copy Files to Target Directory Pending

Now let's copy all the found files to our target directory.

Type this command exactly:
find / -user ira -type f -exec cp -a {} /root/findresults/ \; 2>/dev/null

This command finds all files owned by 'ira' and copies them to the target directory.

Understanding find with -exec

The -exec option allows you to execute a command on each found file:

  • {} - Represents the current file found
  • \; - Terminates the -exec command
  • cp -a - Copy with archive mode (preserves attributes)

Alternative approach: You could also use find / -user ira -exec cp {} /root/findresults/ \;

4 Verify the Results Pending

Finally, let's verify that the files were copied successfully.

Type these commands one by one:
ls -la /root/findresults/

Should show the copied files with their permissions and ownership

ls /root/findresults/ | wc -l

Should show the number of files copied

file /root/findresults/*

Should show file types of the copied files

Verification Commands

After copying files, always verify with these commands:

  • ls -la - Shows detailed file information including ownership
  • wc -l - Counts the number of lines/files
  • file - Determines file type
  • stat - Shows detailed file statistics
