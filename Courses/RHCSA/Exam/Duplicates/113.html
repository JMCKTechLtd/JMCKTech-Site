Welcome to the RHCSA YUM Repository Configuration Lab. In this lab, you will configure your system to use the provided YUM repositories and GPG key.

Follow the steps in order. You must complete each step before moving to the next one.

1 Navigate to YUM Repository Directory Current Step

First, let's navigate to the directory where YUM repository configuration files are stored.

Type this command exactly:
cd /etc/yum.repos.d/

This directory contains all the repository configuration files for YUM.

Understanding YUM Repositories

YUM (Yellowdog Updater Modified) is the package manager for RHEL. Repository files define where packages can be downloaded from and include security settings like GPG keys.

2 Create Repository Configuration File Pending

Now let's create the repository configuration file.

Edit the repository file:
vim /etc/yum.repos.d/redhat.repo

Add the following configuration to the file:

[BaseOS] name=BaseOS baseurl=http://utility.domain15.example.com/BaseOS enabled=1 gpgcheck=1 gpgkey=http://utility.domain15.example.com/RPM-GPG-KEY-redhat-release [AppStream] name=AppStream baseurl=http://utility.domain15.example.com/AppStream enabled=1 gpgcheck=1 gpgkey=http://utility.domain15.example.com/RPM-GPG-KEY-redhat-release

To save and exit vim:

  1. Press ESC to ensure you're in command mode
  2. Type :wq and press Enter to write and quit

Repository Configuration Parameters

  • name: Descriptive name for the repository
  • baseurl: URL where packages are located
  • enabled: 1 to enable, 0 to disable the repository
  • gpgcheck: 1 to verify package signatures
  • gpgkey: URL of the GPG key for verification

3 Clean YUM Cache Pending

After creating the repository file, we need to clean the YUM cache to ensure it reads the new configuration.

Type this command exactly:
yum clean all

This command removes cached data from all enabled repositories, forcing YUM to refresh its metadata.

4 Verify Repository Configuration Pending

Now let's verify that our repositories are configured correctly and working.

Type this command exactly:
yum repolist

This command lists all enabled repositories and shows the number of available packages.

5 List Available Packages Pending

Finally, let's verify that we can list packages from the configured repositories.

Type this command exactly:
yum list all

This command lists all available packages from all enabled repositories.

YUM Verification Commands

After configuring repositories, always verify with these commands:

  • yum repolist - Shows enabled repositories
  • yum list all - Lists all available packages
  • yum search <package> - Searches for specific packages
  • yum info <package> - Shows detailed package information
