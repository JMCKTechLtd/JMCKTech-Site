1 Edit /etc/yum.repos.d/redhat.repo Current Step

Use vim to create or edit /etc/yum.repos.d/redhat.repo so it defines the BaseOS and AppStream repositories with the required URLs and GPG key.

Type this command:
vim /etc/yum.repos.d/redhat.repo

Inside the editor, ensure the file contains at least:

[BaseOS] name=BaseOS baseurl=http://utility.domain15.example.com/BaseOS enabled=1 gpgcheck=1 gpgkey=http://utility.domain15.example.com/RPM-GPG-KEY-redhat-release [AppStream] name=AppStream baseurl=http://utility.domain15.example.com/AppStream enabled=1 gpgcheck=1 gpgkey=http://utility.domain15.example.com/RPM-GPG-KEY-redhat-release

Repo configuration file

Files under /etc/yum.repos.d/ define YUM / DNF repositories. Each section configures a named repo with baseurl, enabled, gpgcheck and gpgkey.

2 Clean YUM metadata Pending

After changing repo configuration, clean any cached metadata so the system uses the new URLs:

yum clean all

3 Verify enabled repositories Pending

Verify that the new repositories are enabled and reachable:

yum repolist

RHCSA-style verification

In the exam, running yum repolist is a quick way to prove that your repository configuration is valid and that required repos are enabled.
