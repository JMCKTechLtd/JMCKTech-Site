Group Management Concepts

groupadd Command Creates a new group with specified GID:
groupadd -g GID groupname
Group Information Stored in /etc/group
Format: groupname:x:GID:members
GID Ranges System groups: 0-999
User groups: 1000-60000
Verification Check group with:
getent group groupname
or grep groupname /etc/group

Step 1: Add Admin Group

Create the admin group with GID 600:

Command to create group:

Type: groupadd -g 600 admin

Understanding Group Creation

The groupadd command creates a new group entry in /etc/group. The -g option specifies the group ID (GID). GIDs below 1000 are typically reserved for system groups.

Step 2: Verify Group Creation

Check that the group was created correctly:

Command to verify group:

Type: getent group admin

Expected output: admin:x:600:

Group Verification

After creating a group, always verify:

  • The group exists in /etc/group
  • It has the correct GID
  • No members are listed unless specified
