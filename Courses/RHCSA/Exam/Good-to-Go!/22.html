Welcome to the RHCSA User and Group Management Lab. In this lab, you will create users and add them to the admin group with specific passwords.

Follow the steps in order. You must complete each step before moving to the next one.

1 Verify Admin Group Exists Current Step

First, let's verify that the admin group exists with GID 600.

Type this command exactly:
grep admin /etc/group

If the admin group doesn't exist, you'll need to create it:

groupadd -g 600 admin

Understanding Group Management

Groups in Linux allow you to manage permissions for multiple users. The admin group with GID 600 is a special administrative group.

2 Create User2 with Admin Group Pending

Now let's create user2 and add them to the admin group as a supplementary group.

Type this command exactly:
useradd -G admin user2

User Creation Options

The -G option specifies supplementary groups for the new user. This adds the user to the admin group in addition to their primary group.

3 Set Password for User2 Pending

Now set the password for user2 to 'redhat'.

Type this command exactly:
passwd user2

When prompted, enter 'redhat' as the password (you'll need to type it twice).

Alternative Password Setting

You can also set the password in one command without interaction:

echo "redhat" | passwd --stdin user2

4 Create User3 with Admin Group Pending

Now let's create user3 and add them to the admin group as well.

Type this command exactly:
useradd -G admin user3

5 Set Password for User3 Pending

Set the password for user3 to 'redhat'.

Type this command exactly:
passwd user3

When prompted, enter 'redhat' as the password (you'll need to type it twice).

6 Verify User Configuration Pending

Finally, let's verify that our configurations are working correctly.

Type these commands one by one:
id user2

Should show admin in the groups list

id user3

Should show admin in the groups list

grep user2 /etc/passwd

Should show user2 entry

grep user3 /etc/passwd

Should show user3 entry

Verification Commands

After creating users and groups, always verify with these commands:

  • id username - Shows user ID and group membership
  • grep username /etc/passwd - Shows user account details
  • grep groupname /etc/group - Shows group membership
