Goal: Use Control Panel → Programs and Features to uninstall Old Antivirus Security Suite.

  1. Open Start and choose Control Panel (or right-click Start ▶ Control Panel).
  2. Select Programs & Features.
  3. Click Old Antivirus Security Suite to select it.
  4. Click Uninstall and confirm.
  5. Wait for success toast. Use Reset Lab to restore the initial state.
