Lab: Entra ID - Add User to Group
Goal: Add Jane Smith to the Marketing Team group.
  1. Navigate to Azure Active Directory admin center from Azure portal
  2. Go to Users > All users in the left navigation
  3. Search for and select Jane Smith
  4. Go to the Groups tab in the user profile
  5. Click + Add to group
  6. Select Marketing Team and click Add
  7. Verify the membership appears in the list and the success toast is shown
