Lab: Entra ID - Reset User Password
Goal: Reset Jane Smith's password and require a password change at next sign-in.
  1. Click Azure Active Directory in the left navigation.
  2. Open Jane Smith's user profile.
  3. Click the Reset password tab.
  4. Enter and confirm a strong password.
  5. Tick Require password change at next sign-in.
  6. Click Reset password and confirm the success toast.
