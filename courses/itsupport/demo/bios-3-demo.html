BIOS Lab 3: BIOS Date/Time Configuration

AUTOMATED DEMO

Lab Guide

Learning Objectives

Scenario: CMOS battery has failed, resetting BIOS settings.

In this lab, you will learn to:

  • Navigate BIOS date/time settings
  • Correct System Time to 09:30:00
  • Correct System Date to 05/09/2025
  • Save BIOS configuration changes

Why this matters: Incorrect system time causes SSL certificate errors, file timestamp issues, and scheduling problems.

Lab Requirements

○ Navigate to time fields
○ Set time to 09:30:00
○ Set date to 05/09/2025
○ Save and exit BIOS

1 Set System Time Current Step

Navigate through time fields (HH:MM:SS) using Tab, adjust values using ↑↓ arrows to set 09:30:00.

2 Set System Date Pending

Navigate to date row using ↑↓, then adjust MM/DD/YYYY to 05/09/2025.

3 Save Changes Pending

Press F10 to save the corrected date/time settings and exit BIOS.
