Lab Guide

Learning Objectives

In this lab, you will learn to:

  • Select in-progress tickets for resolution
  • Add resolution notes and codes
  • Mark tickets as resolved
  • Select resolved tickets for closure
  • Complete the ticket lifecycle

Lab Requirements

○ Select ticket for resolution
○ Add resolution notes
○ Mark ticket as resolved
○ Select ticket for closure
○ Close ticket

1 Select In-Progress Ticket Current Step

Select a ticket with "In Progress" status that is ready for resolution.

2 Open Resolution Modal Pending

Click the "Resolve" button to open the resolution modal.

3 Add Resolution Notes Pending

Enter detailed resolution notes explaining how the issue was fixed.

4 Select Resolved Ticket Pending

Select the newly resolved ticket for final closure.

5 Close Ticket Pending

Complete the ticket lifecycle by closing the resolved ticket.
