Lab Guide

Learning Objectives

In this lab, you will learn to:

  • Select tickets from the incident list
  • Open ticket details for editing
  • Update ticket status and priority
  • Modify assignment information
  • Save changes to ticket records

Lab Requirements

○ Select a ticket from the list
○ Open ticket for editing
○ Update ticket status
○ Change ticket priority
○ Save ticket changes

1 Select Ticket Current Step

Select a ticket from the incidents list by clicking its checkbox.

2 Open for Editing Pending

Click the "Edit Selected" button to open the ticket editing modal.

3 Update Status Pending

Change the ticket status from "Open" to "In Progress" using the status dropdown.

4 Change Priority Pending

Update the ticket priority from "High" to "Medium" in the priority field.

5 Save Changes Pending

Click the "Save" button to apply all changes to the ticket.
