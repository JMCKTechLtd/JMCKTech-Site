Welcome to the RHCSA User & Group Management Lab. Learn to create and manage users and groups.

1 Create New User Current Step

First, let's create a new user account.

Type this command exactly:
sudo useradd -m -s /bin/bash rhcsa_user

Understanding useradd

How it works: useradd creates a user. -m creates home directory, -s sets shell.

2 Set User Password Pending

Now let's set a password for the user.

Set password:
sudo passwd rhcsa_user

Understanding passwd

How it works: passwd sets or changes user passwords. Passwords must meet complexity requirements.

3 Create Group Pending

Now let's create a new group.

Create group:
sudo groupadd rhcsa_group

Understanding groupadd

How it works: groupadd creates a new group. Groups organize users for permission management.

4 Add User to Group Pending

Now let's add the user to the group.

Add to group:
sudo usermod -aG rhcsa_group rhcsa_user

Understanding usermod

How it works: usermod -aG appends user to group. Without -a, it replaces all groups!

5 Verify User & Groups Pending

Finally, let's verify the user and group membership.

Check user info:
id rhcsa_user
Check groups:
groups rhcsa_user

Understanding Verification Commands

How it works: id shows UID, GID, and groups. groups lists all groups for a user.
