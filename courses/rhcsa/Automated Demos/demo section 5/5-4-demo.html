Welcome to the RHCSA Networking Lab. Learn to configure network settings using nmcli and other tools.

1 View IP Addresses Current Step

First, let's view the current IP address configuration.

Type this command exactly:
ip addr show

Understanding ip Command

How it works: ip addr show displays all network interfaces and their IP addresses. Use ip a as a shortcut.

2 View Network Connections Pending

Now let's view NetworkManager connections.

View connections:
nmcli connection show

Understanding nmcli

How it works: NetworkManager CLI (nmcli) manages network connections. nmcli con show lists all connections.

3 Configure Static IP Pending

Now let's configure a static IP address.

Set static IP:
nmcli connection modify "Wired connection 1" ipv4.addresses 192.168.1.100/24
Activate connection:
nmcli connection up "Wired connection 1"

Understanding Static IP Configuration

How it works: nmcli con mod modifies connection settings. Changes require nmcli con up to activate.

4 Configure Hostname Pending

Now let's set the system hostname.

Set hostname:
hostnamectl set-hostname rhcsa-lab
Verify:
hostnamectl status

Understanding hostnamectl

How it works: hostnamectl manages system hostname. Changes persist across reboots.

5 Test Connectivity Pending

Finally, let's test network connectivity.

Ping Google DNS:
ping -c 4 8.8.8.8

Understanding ping

How it works: ping tests network connectivity. The -c flag limits packet count.
