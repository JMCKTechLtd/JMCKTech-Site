Welcome to the RHCSA Software Management Lab. Learn to manage packages with rpm, dnf, and yum.

1 Query Installed Packages Current Step

First, let's query installed packages using rpm.

Type this command exactly:
rpm -qa | grep bash

Understanding rpm

How it works: rpm -qa lists all installed packages. rpm -qi shows package info, rpm -ql lists files.

2 Search for Packages Pending

Now let's search for packages in repositories.

Search for nano:
dnf search nano

Understanding dnf search

How it works: dnf search searches package names and descriptions in configured repositories.

3 Install Package Pending

Now let's install the nano editor.

Install nano:
sudo dnf install -y nano

Understanding dnf install

How it works: dnf install downloads and installs packages. The -y flag auto-confirms prompts.

4 Remove Package Pending

Now let's remove the package we just installed.

Remove nano:
sudo dnf remove -y nano

Understanding dnf remove

How it works: dnf remove uninstalls packages and their dependencies (unless needed by other packages).

5 Update System Pending

Finally, let's update all packages on the system.

Update system:
sudo dnf update -y

Understanding dnf update

How it works: dnf update updates all packages to their latest versions. Use dnf upgrade to also remove obsolete packages.
