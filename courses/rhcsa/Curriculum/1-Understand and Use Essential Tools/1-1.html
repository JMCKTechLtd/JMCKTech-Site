RHCSA Lab 1.1: Shell Prompt & Commands

Lab Instructions

Step 1: The Shell Prompt

Type exactly: echo "Hello, RHCSA!"

Understanding the Shell Prompt

What Your command-line interface (e.g., bash)
Why Indicates the shell is ready for commands
Where Appears in terminal emulators
When After login or opening a terminal

How it works: The prompt (e.g., [user@host ~]$) shows username, hostname, and current directory. The echo command outputs text to the terminal.

Step 2: Command Syntax

Type exactly: ls -l /home

Understanding Command Structure

What command [options] [arguments]
Why Options modify behavior, arguments specify targets
Where Used in every shell interaction
When When executing any command

How it works: ls lists files, -l enables long format, and /home specifies the directory. Options typically begin with - (short) or -- (long).

Step 3: Tab Completion

Type exactly: cd /etc (use Tab after cd /e)

Then type: ls

Understanding Tab Completion

What Automatic command/path completion
Why Saves time and prevents typos
Where Works in most modern shells
When While typing commands/paths

How it works: Pressing Tab attempts to complete the current word. If multiple matches exist, press Tab twice to see options. The shell searches PATH for commands and filesystem for paths.
