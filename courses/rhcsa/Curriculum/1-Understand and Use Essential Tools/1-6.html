RHCSA Lab 1.6: Archive & Compress Files

Lab Instructions

Step 1: Create a Tar Archive

Type exactly: tar -cvf archive.tar file1.txt file2.txt

Understanding Tar Archives

What Combines multiple files into one archive file
Why Easier file management and transfer
Where Used for backups and software distribution
When When you need to bundle files together

How it works: The tar command (tape archive) creates (-c), shows verbose output (-v), and specifies the filename (-f). The files file1.txt and file2.txt will be included in the archive.

file1.txt
file2.txt
archive.tar (will be created)

Step 2: List Archive Contents

Type exactly: tar -tvf archive.tar

Understanding Archive Inspection

What View contents without extracting
Why Verify archive contents before extraction
Where Useful for checking downloaded archives
When Before extracting any archive

How it works: The -t option lists contents, -v shows details, and -f specifies the file. This is a safe way to inspect archives from untrusted sources.

Step 3: Extract Archive

Type exactly: mkdir extracted

Then type: tar -xvf archive.tar -C extracted

Understanding Archive Extraction

What Restores files from archive
Why Access original files from archive
Where Common in software installation
When When you need the original files

How it works: The -x option extracts, -v shows progress, -f specifies the file, and -C changes to the target directory. Always extract to a specific directory to avoid clutter.

Step 4: Compress with Gzip

Type exactly: gzip archive.tar

Then type: gunzip archive.tar.gz

Understanding Gzip Compression

What Common compression tool (.gz)
Why Reduce file size for storage/transfer
Where Used for log files, web assets
When When file size matters more than CPU

How it works: gzip compresses files (creates .gz), while gunzip decompresses. Gzip is fast but doesn't provide the best compression ratio compared to newer algorithms.

Step 5: Create Compressed Tar Archives

Type exactly: tar -czvf archive.tar.gz file1.txt file2.txt

Then type: tar -cjvf archive.tar.bz2 file1.txt file2.txt

Finally type: tar -cJvf archive.tar.xz file1.txt file2.txt

Understanding Compression Options

What Different compression algorithms
Why Balance between speed and ratio
Where Choose based on requirements
When gzip: fast, bzip2: balanced, xz: best ratio

How it works: The -z uses gzip, -j uses bzip2, and -J uses xz. Each has tradeoffs: gzip is fastest, bzip2 balances speed/ratio, xz offers best compression but is slowest.

gzip: archive.tar.gz
bzip2: archive.tar.bz2
xz: archive.tar.xz
