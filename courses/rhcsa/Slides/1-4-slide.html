3 Creating and Copying Files Current Step

Now let's learn how to create and copy files using touch and cp commands.

Type these commands exactly: 
touch newfile.txt
cp newfile.txt copied_file.txt
cp -r documents documents_backup
Understanding touch and cp Commands:
  • touch creates empty files or updates timestamps
  • cp copies files/directories
  • Use -r option for directories

4 Moving and Renaming Files Current Step

Now let's learn how to move and rename files using the mv command.

Type these commands exactly: 
mv copied_file.txt documents/
mv documents/copied_file.txt documents/renamed_file.txt
Understanding mv Command:
  • mv moves or renames files/directories
  • Same command as moving within same directory
  • Overwrites existing files without warning
  • Use -i for interactive mode
[user@lab ~]$ touch newfile.txt Created file: newfile.txt [user@lab ~]$ cp newfile.txt copied_file.txt Copied newfile.txt to copied_file.txt [user@lab ~]$ cp -r documents documents_backup Copied directory documents to documents_backup Step 3 completed. Moving to step 4. [user@lab ~]$ Type: mv copied_file.txt documents/

5 Removing Files and Directories Current Step

Now let's learn how to remove files and directories using the rm command.

Type these commands exactly: 
rm file1.txt
rm -r documents_backup
Understanding rm Command:
  • rm removes files/directories
  • Permanent deletion (no trash bin)
  • Use -r for directories
  • Use -i for interactive mode
  • Be careful with wildcards (rm *)
[user@lab ~]$ mv copied_file.txt documents/ Moved copied_file.txt to documents/ [user@lab ~]$ mv documents/copied_file.txt documents/renamed_file.txt Moved documents/copied_file.txt to documents/renamed_file.txt Step 4 completed. Moving to step 5. [user@lab ~]$ Type: rm file1.txt

6 Finding Files Current Step

Finally, let's learn how to find files using the find command.

Type these commands exactly: 
find . -name "*.txt"
find /etc -type f -name "*.conf"
Understanding find Command:
  • find searches for files/directories
  • Can search by name, type, size, time, etc.
  • Basic syntax: find [path] [criteria] [action]
  • Searches recursively through directories
[user@lab ~]$ rm file1.txt Removed file1.txt [user@lab ~]$ rm -r documents_backup Removed directory documents_backup Step 5 completed. Moving to step 6. [user@lab ~]$ Type: find . -name "*.txt"
