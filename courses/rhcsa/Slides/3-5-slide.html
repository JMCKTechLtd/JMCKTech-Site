Lab Instructions

Lab Objective
Learn how to manage swap space on a Red Hat Enterprise Linux system. You will create a swap file, activate and deactivate it, and make it persistent across reboots.

2 Creating a Swap File Current Step

Now let's create a 512MB swap file.

Type the commands shown below:

sudo fallocate -l 512M /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile

Now let's activate and deactivate the swap file.

4 Making Swap Persistent Current Step

Finally, let's make the swap file persistent across reboots.

Type the commands shown below:

echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
sudo swapon -a
swapon --show
