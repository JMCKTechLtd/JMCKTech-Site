Lab Instructions

Lab Objective
Learn to create and manage archive files using tar, and compress them using different compression algorithms (gzip, bzip2, xz).

Time Estimate: 15–20 minutes
0% Complete

Archive & Compression Requirements

Create tar archive: tar -cvf archive.tar file1.txt file2.txt
List archive contents: tar -tvf archive.tar
Extract archive to directory: mkdir extracted then tar -xvf archive.tar -C extracted
Compress with gzip: gzip archive.tar then gunzip archive.tar.gz
Create compressed tar archives: tar -czvf archive.tar.gz, tar -cjvf archive.tar.bz2, tar -cJvf archive.tar.xz

5 Create Compressed Tar Archives Current Step

Finally, let's create compressed tar archives using different compression algorithms.

Create gzip compressed archive:

tar -czvf archive.tar.gz file1.txt file2.txt

Create bzip2 compressed archive:

tar -cjvf archive.tar.bz2 file1.txt file2.txt

Create xz compressed archive:

tar -cJvf archive.tar.xz file1.txt file2.txt
What Different compression algorithms
Why Balance between speed and ratio
Where Choose based on requirements
When gzip: fast, bzip2: balanced, xz: best ratio

How it works: The -z uses gzip, -j uses bzip2, and -J uses xz.

[user@lab ~]$ gzip archive.tar Compressed archive.tar to archive.tar.gz [user@lab ~]$ gunzip archive.tar.gz Decompressed archive.tar.gz to archive.tar Step 4 completed. Gzip compression tested.
