Welcome to the RHCSA User Management Lab. In this lab, you will create and manage user accounts on a Red Hat Enterprise Linux system.

Follow the steps in order. You must complete each step before moving to the next one.

1 Create the User Current Step

Create a new user named alex with UID 1234:

Type this command exactly:
useradd -u 1234 alex

Understanding User Creation

The useradd command creates new user accounts. The -u option specifies the user ID (UID), which must be a unique number. UIDs below 1000 are typically reserved for system accounts.

2 Set the Password Pending

Set the password for user alex to alex111:

Method 1: Interactive password setting
passwd alex

Then enter alex111 twice when prompted

Method 2: Non-interactive password setting
echo "alex111" | passwd --stdin alex

Understanding Password Management

The passwd command changes user passwords. The --stdin option allows setting the password non-interactively, which is useful for scripts. Note that this method may show the password in process listings temporarily.

3 Verify the User Pending

Verify that the user was created correctly and the password works:

Check user information:
id alex

Should show: uid=1234(alex)

getent passwd alex

Should show user details

Test the password:
su - alex

Enter password alex111 when prompted

Type exit to return to root

Verification Commands

After creating a user, always verify with these commands:

  • id username - Shows UID, GID, and groups
  • getent passwd username - Checks user database entry
  • su - username - Tests password authentication
  • cat /etc/passwd | grep username - Alternative check
