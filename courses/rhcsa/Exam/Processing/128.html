Follow each step in order. The simulated terminal on the right will validate your commands.

1 Run the find command for user alex Current Step

Use find to locate all regular files owned by user alex, starting at the root directory /, and redirect the output to /home/alex/files.

Type one of these equivalent commands:
find / -user alex -type f >/home/alex/files
find / -user alex -type f > /home/alex/files

find with ownership and type filters

-user alex limits results to files owned by alex, and -type f restricts matches to regular files only. The redirection writes all matching paths into a single file.

2 Verify /home/alex/files exists Pending

Confirm that the report file /home/alex/files was created by the find command.

ls -l /home/alex/files

3 Inspect the contents of /home/alex/files Pending

View the beginning of the file to confirm that it contains a list of file paths owned by alex.

head /home/alex/files

RHCSA-style verification

In the exam, examiners often check that the target file exists and contains appropriate output. A quick head or cat is usually enough.
