Kernel Upgrade Concepts

Kernel Packages Typically includes:
kernel-version.rpm - Main kernel package
kernel-firmware-version.rpm - Firmware files
Installation Use rpm -ivh to install new kernel
Never use -U as it removes old kernel
Multiple kernels can coexist
GRUB Configuration Located at /boot/grub/grub.conf
default= sets default kernel
Count starts at 0 for first entry
Verification uname -r shows running kernel
rpm -q kernel shows installed kernels
Reboot to test new kernel

Step 1: Download the New Kernel

Download the kernel package:

Type: wget ftp://server1.domain10.example.com/pub/update/new.kernel

Or: curl -O ftp://server1.domain10.example.com/pub/update/new.kernel

Understanding Kernel Packages

The kernel package contains the core operating system files. Downloading from the specified FTP server ensures you get the correct, tested version for your system.

Step 2: Install the New Kernel

Install the kernel package:

Type: rpm -ivh kernel-2.6.32-71.7.1.el6.x86_64.rpm kernel-firmware-2.6.32-71.7.1.el6.noarch.rpm

Expected output:

Preparing... ########################################### [100%] 1:kernel-firmware ########################################### [50%] 2:kernel ########################################### [100%]

Understanding Kernel Installation

Using rpm -ivh installs the new kernel alongside the existing one. The -i flag means install, -v is verbose, and -h shows progress bars.

Step 3: Configure GRUB

Verify GRUB configuration:

Type: cat /boot/grub/grub.conf

Ensure it contains these settings:

default=0 title Red Hat Enterprise Linux Server (2.6.32-71.7.1.el6.x86_64) root (hd0,0) kernel /vmlinuz-2.6.32-71.7.1.el6.x86_64 ro root=/dev/mapper/vol0-root rd_LVM_LV=vol0/root rd_NO_LUKS rd_NO_MD rd_NO_DM LANG=en_US.UTF-8 SYSFONT=latarcyrheb-sun16 KEYBOARDTYPE=pc KEYTABLE=us crashkernel=auto rhgb quiet initrd /initramfs-2.6.32-71.7.1.el6.x86_64.img

Understanding GRUB Configuration

GRUB (Grand Unified Bootloader) controls which kernel boots by default. The default=0 line means the first kernel entry will boot automatically. Each kernel has its own title section.
