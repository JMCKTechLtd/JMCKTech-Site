Step 1: Edit Natasha's Crontab

Edit cron jobs for user natasha:

Type: crontab -e -u natasha

Add the following line:

23 14 * * * /bin/echo hiya

Cron Time Format

The time specification 23 14 * * * means:

  • Minute: 23
  • Hour: 14 (2:00 PM)
  • Every day of the month (*)
  • Every month (*)
  • Every day of the week (*)

Step 2: Verify the Cron Job

List natasha's cron jobs:

Type: crontab -l -u natasha

Should show:

23 14 * * * /bin/echo hiya

Step 3: Ensure crond Service is Running

Enable and start crond service:

Type: systemctl enable crond

Type: systemctl restart crond

About crond Service

The crond service is responsible for executing scheduled commands (cron jobs). It must be running for any cron jobs to execute.
