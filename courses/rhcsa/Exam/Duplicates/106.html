Welcome to the RHCSA User Management Lab. In this lab, you will create a user account with specific requirements.

Follow the steps in order. You must complete each step before moving to the next one.

1 Create User with Specific UID Current Step

First, let's create a user named alex with UID 3400.

Type this command exactly:
useradd -u 3400 alex

Understanding User Creation

The useradd command creates a new user account. The -u option specifies the user ID (UID).

UIDs below 1000 are typically reserved for system accounts, while UIDs 1000 and above are for regular users.

2 Set User Password Pending

Now let's set the password for user alex to redhat.

Type this command exactly:
passwd alex

When prompted, enter redhat as the password (twice).

Understanding Password Management

The passwd command changes a user's password. You need to enter the new password twice for confirmation.

In a real environment, you would use a more secure password, but for this lab exercise, we're using a simple one.

3 Verify User Creation Pending

Finally, let's verify that the user was created correctly with the right UID.

Type this command exactly:
id alex

This should show that user alex has UID 3400.

Verification Commands

After creating users, always verify with these commands:

  • id username - Shows UID, GID, and groups for a user
  • cat /etc/passwd | grep username - Shows user account information
  • getent passwd username - Queries user database
