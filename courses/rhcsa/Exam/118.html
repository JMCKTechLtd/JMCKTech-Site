Welcome to the RHCSA File Search and Processing Lab. In this lab, you will use the find command to locate files based on ownership and size criteria.

Follow the steps in order. You must complete each step before moving to the next one.

1 Find Files Owned by User Natasha Current Step

First, let's find all files owned by the user natasha and redirect the output to a file.

Type this command exactly:
find / -user natasha -type f > /home/alex/files

This command:

  • Searches the entire filesystem (/)
  • Finds files owned by user natasha
  • Limits to regular files (-type f)
  • Redirects output to /home/alex/files

Understanding the find Command

The find command is a powerful tool for searching files. Key options:

  • -user username - Find files owned by specific user
  • -type f - Find only regular files (not directories)
  • -size +5M - Find files larger than 5MB
  • -name pattern - Find files by name pattern

2 Verify Natasha Files Output Pending

Now let's verify that the files owned by natasha were found and saved correctly.

Type this command exactly:
cat /home/alex/files

This should display the list of files owned by user natasha.

You can also check how many files were found:
wc -l /home/alex/files

This will show the number of files found.

3 Create the /find Directory Pending

Now let's create the directory where we'll store the large files list.

Type this command exactly:
mkdir /find

This creates the /find directory for storing our search results.

Directory Creation

The mkdir command creates directories. Useful options:

  • mkdir dirname - Create a single directory
  • mkdir -p /path/to/dir - Create parent directories if needed
  • mkdir -m 755 dirname - Create with specific permissions

4 Find Large Files in /etc Directory Pending

Now let's find files larger than 5MiB in the /etc directory and save the results.

Type this command exactly:
find /etc -size +5M > /find/largefiles

This command:

  • Searches in the /etc directory
  • Finds files larger than 5MiB (-size +5M)
  • Redirects output to /find/largefiles

File Size Searching

The -size option in find supports various units:

  • c - bytes
  • k - kilobytes
  • M - megabytes
  • G - gigabytes
  • + - greater than
  • - - less than

Example: -size +10M finds files larger than 10MB.

5 Verify Large Files Output Pending

Finally, let's verify that the large files were found and saved correctly.

Type these commands to verify:
cat /find/largefiles

Displays the list of large files found in /etc.

ls -lh /find/largefiles

Shows information about the output file itself.

du -sh /etc

Shows the total size of the /etc directory for comparison.

File Size Commands

Other useful commands for file size analysis:

  • du -sh directory - Show total size of directory
  • ls -lh file - Show file size in human-readable format
  • stat file - Show detailed file information including size
