Swap Space Concepts

Swap Space Extension of physical memory that uses disk space. Used when RAM is full or for hibernation.
mkswap Initializes a partition or file as swap space. Must be done before first use.
swapon/swapoff Activates/deactivates swap space. swapon -a activates all in fstab.
/etc/fstab Contains filesystem mount information. Swap entries use 'swap' as mount point.

Step 1: Create the Swap Partition

Start fdisk on the disk:

Type: fdisk -cu /dev/vda

Then follow these steps:

n (create new partition) p (primary partition) [Enter] (accept default partition number) [Enter] (accept default first sector) +500M (set partition size) t (change partition type) 82 (Linux swap type) w (write changes and exit)

Partitioning Considerations

When creating a new partition:

  • Ensure you don't modify existing partitions
  • Set the correct partition type (82 for Linux swap)
  • Verify there's enough space on the disk

Step 2: Initialize Swap Space

Initialize the new partition as swap:

Type: mkswap /dev/vdaX (replace X with your partition number)

About mkswap

The mkswap command writes a swap signature to the partition. This only needs to be done once when first creating the swap space.

Step 3: Activate the Swap Space

Activate the new swap partition:

Type: swapon /dev/vdaX

Verify swap is active:

Type: swapon --show or free -h

Temporary vs Persistent Activation

swapon only activates swap temporarily. To make it persistent across reboots, you must add an entry to /etc/fstab.

Step 4: Configure Persistent Mount

Edit /etc/fstab:

Type: vim /etc/fstab

Add this line (replace X with your partition number):

/dev/vdaX swap swap defaults 0 0

Verify fstab entry:

Type: mount -a (tests fstab without reboot)

Type: swapon --show to verify swap is still active

fstab Format for Swap

The fstab entry for swap has special fields:

  • First field: Device (/dev/vdaX or UUID)
  • Second field: 'swap' (instead of mount point)
  • Third field: 'swap' as filesystem type
  • Options, dump, and pass fields same as other filesystems
