Step 1: Download Kernel Update

Method 1: Manual Download

Access the URL in a browser or use wget:

Type: wget http://server.domain11.example.com/pub/updates/kernel-*.rpm

Method 2: YUM Repository

Add the repository and install directly:

yum-config-manager --add-repo=http://content.example.com/rhel7.0/x86-64/errata yum clean all yum list kernel

Understanding Kernel Updates

The kernel is the core of the operating system. Updates may include security patches, bug fixes, and new features. Always verify the source of kernel updates before installation.

Step 2: Install Kernel Package

Manual Installation:

Type: rpm -ivh kernel-*.rpm

YUM Installation:

Type: yum -y install kernel

Kernel Installation Notes

When installing a new kernel:

  • The old kernel remains available as a fallback
  • GRUB2 automatically detects new kernels
  • Verify installation with rpm -q kernel

Step 3: Configure GRUB2

Check current GRUB2 default:

Type: grub2-editenv list

Set new kernel as default:

Type: grub2-set-default 'kernel full name'

Update GRUB2 configuration:

Type: grub2-mkconfig -o /boot/grub2/grub.cfg

GRUB2 Configuration

GRUB2 is the bootloader that loads the kernel. Key points:

  • Configuration files are in /boot/grub2/
  • Always keep at least one old kernel as backup
  • Changes take effect after reboot

Step 4: Verification

Check current kernel version:

Type: uname -r

List installed kernels:

Type: rpm -q kernel

Verify GRUB2 configuration:

Type: cat /boot/grub2/grub.cfg | grep menuentry

Verification Steps

After kernel upgrade, verify:

  • New kernel is the default in GRUB2
  • Old kernel is still present
  • System boots correctly with new kernel
