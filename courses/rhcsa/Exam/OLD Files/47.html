YUM Repository Configuration

Repository Files YUM repositories are configured in .repo files in /etc/yum.repos.d/
Key Parameters [repository-id]: Unique repository ID
name: Human-readable name
baseurl: URL to repository
gpgcheck: Enable/disable GPG checking
Verification yum repolist: List configured repositories
yum list: Show available packages
HTTP Repository For HTTP repositories, ensure network connectivity and correct URL path to repository data

Step 1: Create Repository File

Create a YUM repository configuration file:

Navigate to yum repositories directory:

Type: cd /etc/yum.repos.d/

Create or edit the repository file:

Type: vim base.repo

Ensure the file contains these settings:

[base] name=base baseurl=http://instructor.example.com/pub/rhel6/dvd gpgcheck=0

Understanding YUM Repositories

YUM repositories provide a way to install and update packages from a central location. Key points:

  • Repository files must have .repo extension
  • gpgcheck=0 disables package verification (not recommended for production)
  • The baseurl must point to a valid repository with proper metadata

Step 2: Verify Repository

List available packages:

Type: yum list

Check repository status:

Type: yum repolist (should show 'base' repository)

Repository Verification

After configuring a repository, verify it works with these commands:

  • yum repolist - Shows enabled repositories
  • yum list - Displays available packages
  • yum search [package] - Searches for specific packages
