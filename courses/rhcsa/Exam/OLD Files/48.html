Network Gateway Configuration

Gateway Configuration The gateway defines the route for packets leaving the local network. Can be set in either the network file or interface configuration.
Network File /etc/sysconfig/network - System-wide network configuration
Contains GATEWAY=192.168.0.254
Interface File /etc/sysconfig/network-scripts/ifcfg-eth0 - Interface-specific settings
Contains GATEWAY=192.168.0.254
Verification ip route show - Shows default gateway
ping 192.168.1.1 - Tests connectivity to other network

Step 1: Configure Gateway in Network File

Edit the system-wide network configuration file:

Edit the network configuration file:

Type: vim /etc/sysconfig/network

Ensure the file contains:

GATEWAY=192.168.0.254

Understanding the Network File

The /etc/sysconfig/network file contains system-wide network parameters. The gateway specified here will be used as the default route unless overridden in interface-specific configuration.

Step 2: Configure Gateway in Interface File

Edit the network interface configuration file:

Navigate to network scripts directory:

Type: cd /etc/sysconfig/network-scripts/

Edit the interface configuration:

Type: vim ifcfg-eth0

Ensure the file contains these settings:

DEVICE=eth0 BOOTPROTO=static ONBOOT=yes IPADDR=192.168.0.? NETMASK=255.255.255.0 GATEWAY=192.168.0.254

Understanding Interface Configuration

The interface configuration file controls how your network interface behaves. Key parameters:

  • BOOTPROTO=static: Uses static IP configuration
  • GATEWAY: Defines the default gateway for this interface
  • ONBOOT=yes: Brings interface up at boot

Step 3: Apply Changes

Restart network service:

Type: systemctl restart network

Verify gateway configuration:

Type: ip route show (should show gateway 192.168.0.254)

Test connectivity:

Type: ping -c 3 192.168.1.1 (should receive replies)

Network Verification

After making changes, always verify with these commands:

  • ip route - Shows routing table including gateway
  • ping - Tests connectivity to other hosts
  • ip addr - Displays IP addresses
