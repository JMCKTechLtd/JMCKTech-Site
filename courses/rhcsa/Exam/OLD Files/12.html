Autofs Configuration Concepts

autofs Service Automatically mounts filesystems when accessed and unmounts them after a period of inactivity.
auto.master The master map that defines the mount points and their corresponding map files.
Map Files Contain the details of what to mount where, with options like read-write, soft mount, etc.
Wildcard Mounts The & symbol in map files allows mounting home directories for all users.

Step 1: Enable and Start autofs Service

Enable autofs to start at boot:

Type: systemctl enable autofs

Start the autofs service:

Type: systemctl start autofs

Understanding autofs Service

The autofs service manages automatic mounting of filesystems. Unlike static mounts in /etc/fstab, autofs mounts are triggered when accessed and automatically unmounted after a period of inactivity.

Step 2: Configure auto.master

Edit the auto.master file:

Type: vim /etc/auto.master

Add the following line:

/rhome /etc/auto.ldap

Understanding auto.master

The auto.master file defines the base mount point (/rhome) and the map file (/etc/auto.ldap) that contains the details of what to mount there.

Step 3: Create auto.ldap Map File

Copy from auto.misc as a template:

Type: cp /etc/auto.misc /etc/auto.ldap

Edit the auto.ldap file:

Type: vim /etc/auto.ldap

Add the following line:

ldapuser40 -rw,soft,intr 172.24.40.10:/rhome/ldapuser40 * -rw,soft,intr 172.24.40.10:/rhome/&

Understanding Map File Syntax

The map file specifies:

  • ldapuser40: Specific mount for this user
  • *: Wildcard for all other users
  • -rw,soft,intr: Mount options (read-write, soft mount, interruptible)
  • 172.24.40.10:/rhome/ldapuser40: NFS server and export path
  • &: Replaces with the username when mounting

Step 4: Restart and Verify

Restart the autofs service:

Type: systemctl restart autofs

Verify the NFS export is available:

Type: showmount -e 172.24.40.10

Test the mount:

Type: su - ldapuser40

Then check if the home directory is accessible.

Verification Steps

After configuration:

  • Use showmount to verify NFS exports
  • Switch to ldapuser40 to test the mount
  • Check /rhome directory for mounted home directories
  • Verify autofs service is running with systemctl status autofs
