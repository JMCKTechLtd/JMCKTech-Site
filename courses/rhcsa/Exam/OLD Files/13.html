Time Synchronization Concepts

chrony The modern time synchronization daemon that replaces ntpd, better suited for systems that are intermittently connected.
NTP Server Network Time Protocol servers provide accurate time references. In this lab, we use 172.24.40.10.
Configuration The main configuration file is /etc/chrony.conf where NTP servers are specified.
Verification Use chronyc sources to check synchronization status and timedatectl for current time settings.

Step 1: Install chrony (if needed)

Check if chrony is installed:

Type: rpm -q chrony

Install chrony if not present:

Type: yum install -y chrony

About chrony

chrony is the default time synchronization service on modern RHEL systems. It's more accurate than ntpd, especially for systems that are not always connected to the network.

Step 2: Configure chrony

Edit the chrony configuration:

Type: vim /etc/chrony.conf

Add or modify the server line:

server 172.24.40.10 iburst

chrony.conf Options

Key configuration options:

  • server: Specifies the NTP server address
  • iburst: Sends a burst of packets initially for faster synchronization
  • allow: Configures which clients can access this server
  • driftfile: Records clock drift rate

Step 3: Enable and Start chronyd

Enable chronyd to start at boot:

Type: systemctl enable chronyd

Start the chronyd service:

Type: systemctl start chronyd

Restart if already running:

Type: systemctl restart chronyd

chronyd Service

The chronyd daemon runs in the background to continuously adjust the system clock. It needs to be running for time synchronization to work.

Step 4: Verify Time Synchronization

Check synchronization sources:

Type: chronyc sources

Check tracking information:

Type: chronyc tracking

Check current time status:

Type: timedatectl

Verification Commands

Key verification commands:

  • chronyc sources: Shows NTP servers being used
  • chronyc tracking: Displays detailed synchronization info
  • timedatectl: Shows current time, timezone, and NTP status
  • chronyc makestep: Forces immediate time correction
